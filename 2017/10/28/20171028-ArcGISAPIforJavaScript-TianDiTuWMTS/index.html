本文介绍了ArcGIS API for JavaScript如何加载天地图WMTS图层，包括地理坐标系、投影坐标系下的矢量和影像地图。读完本文可以使用同样方法加载高德、百度等地图服务。

附：天地图类型

  • vec: 矢量底图
  • cva: 矢量注记图层
  • eva: 矢量注记图层-英文注记
  • img: 影像底图
  • cia: 影像注记图层
  • eia: 影像注记图层 -英文注记
  • ter: 地形底图
  • cta: 地形注记图层
